# Data Table

The Data Table module provides a fully featured, accessible data table for displaying structured information. It is composed from LibyUI primitives and supports sorting, filtering, pagination, and row selection out of the box.

## Key concepts

  <AccordionItem value="item-1">
    <AccordionTrigger>Column definitions</AccordionTrigger>
    <AccordionContent>
      Columns are defined as configuration objects â€” you specify the key, a header label, an optional render function, and sort/filter behaviour per column. This keeps table markup clean and data-driven.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-2">
    <AccordionTrigger>Sorting</AccordionTrigger>
    <AccordionContent>
      Click any sortable column header to toggle ascending/descending order. Sort state is controlled and can be lifted to the page level for URL-synced sorting.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-3">
    <AccordionTrigger>Filtering</AccordionTrigger>
    <AccordionContent>
      Pass a `filter` value and the table handles client-side filtering across all searchable columns. For server-side filtering, pass a controlled `onFilterChange` callback and handle the query in your data fetching layer.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-4">
    <AccordionTrigger>Pagination</AccordionTrigger>
    <AccordionContent>
      The table includes a built-in pagination bar. Configure `pageSize` and `totalRows` to enable server-side pagination, or leave them off for automatic client-side paging.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-5">
    <AccordionTrigger>Row selection</AccordionTrigger>
    <AccordionContent>
      Enable `selectable` to add checkboxes to each row and a "select all" header checkbox. Selected row IDs are available via the `onSelectionChange` callback.
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Basic Usage

```tsx
const columns = [
  { key: "name",   header: "Name",  sortable: true },
  { key: "email",  header: "Email", sortable: true },
  { key: "status", header: "Status" },
];

const data = [
  { id: "1", name: "Jane Smith",  email: "jane@example.com",  status: "Active" },
  { id: "2", name: "John Doe",    email: "john@example.com",  status: "Inactive" },
];

// Pass columns and data to your DataTable component
```
