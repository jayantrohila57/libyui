# Badge

A small badge component for displaying status information or counts.

## Usage

```tsx
import { Badge } from 'libyui'

export default function Example() {
  return (
    <Badge variant="primary">New</Badge>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"primary" \| "secondary" \| "success" \| "warning" \| "error"` | `"primary"` | Badge style |
| `children` | `ReactNode` | - | Badge content |
| `className` | `string` | - | Additional CSS classes |
