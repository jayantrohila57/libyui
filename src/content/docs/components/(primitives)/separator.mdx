## Usage

```tsx
import { Separator } from "@/components/ui/separator";

export default function Example() {
  return (
    <div>
      <div>Content above</div>
      <Separator />
      <div>Content below</div>
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the separator |
| `decorative` | `boolean` | `true` | Whether the separator is decorative (not announced by screen readers) |
| `className` | `string` | - | Additional CSS classes to apply to the separator |

All standard Radix UI Separator Root props are also supported.
